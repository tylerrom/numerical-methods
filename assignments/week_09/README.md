# Week 9: Ordinary and Partial Differential Equations

**Due:** April 8, 2026

This week, you will work on classic problems in physics that involve ordinary differential equations (ODEs). You will use `scipy.integrate.solve_ivp` to solve initial value problems (IVPs), and `scipy.integrate.solve_bvp` to solve boundary value problems (BVPs). If you choose to complete the advanced extension, you will also solve the time-dependent heat equation, which is a partial differential equation (PDE).
